SOAP病历智能评分系统

【主观 (Subjective)】

  • 格式规范性：5/5 → 病历格式规范，按健康问题逐条描述。
  • 主诉完整性：2/2 → 主诉中症状（头痛）、时间（2个月）、程度（加重伴头晕2天）完整。
  • 症状演变：5/5 → 症状演变过程清晰，描述了头痛的频率、持续时间以及治疗反应。详细记录了症状的加重及伴随症状。
  • 诊治经过：3/3 → 详细记录了患者已接受的诊疗情况，包括高血压的诊断、治疗及药物停用原因。
  • 相关病史：3/3 → 既往史完整，详细记录了高血压的诊断和治疗情况，以及停药原因。无糖尿病、冠心病病史。
  • 家族史：2/2 → 家族史描述完整，提到了直系亲属的疾病史。
  • 社会心理：5/5 → 详细描述了患者的生活方式（吸烟、饮食习惯、运动情况）和心理状态（与家人争执）。

主观评分总分 = 5 + 2 + 5 + 3 + 3 + 2 + 5 = 25/25分

【客观 (Objective)】

  • 体格检查：7/8 → 体格检查记录较为完整，包括了体温、脉搏、血压等生命体征，以及神志、精神状态、情绪等评估。同时，记录了重要体征如左侧锁骨下可闻及血管杂音、腹部查体未见异常等。但未详细记录四肢及神经系统检查，扣1分。
  • 实验室检查：4/5 → 实验室检查记录较为完整，包括血常规、尿常规、肝功能、肌酐等关键项目。但未记录血脂、血糖等其他可能重要的检查结果，扣1分。
  • 其他评估：2/2 → 心理评估记录显示患者家庭沟通尚可，工作无压力，社会关系和谐，无精神异常，无心理问题。评估较为全面，无扣分。

客观评分总分 = 7 + 4 + 2 = 13/15分

【评估 (Assessment)】

  • 主要诊断：8/8 → 诊断符合ICD-10标准，明确指出高血压性头痛和原发性高血压2级（极高危）。
  • 危险因素：9/10 → 涵盖了生物维度（肥胖、吸烟、动脉粥样硬化高危因素、锁骨下血管杂音提示动脉狭窄）和心理社会维度（夜间打鼾，提示可能的阻塞性睡眠呼吸暂停综合征），但未提及家庭和社会支持情况。
  • 并发症：3/4 → 识别了潜在的肾动脉狭窄和阻塞性睡眠呼吸暂停综合征，但未提及其他可能的并发症。
  • 依从性：1/2 → 未具体评估患者对治疗的配合度，仅提及了吸烟和肥胖等生活习惯。
  • 家庭资源：0/1 → 未分析家庭支持能力。

评价评分总分 = 8 + 9 + 3 + 1 + 0 = 21/25分

【计划 (Plan)】

  • 检查计划：6/6 → 检查计划涵盖了肾动脉超声、心电图、胸片、尿微量白蛋白、肾功能、经颅超声多普勒、颈动脉及锁骨下动脉超声等必要检查项目，并说明了检查的临床意义。符合评分标准，得满分。
  • 药物治疗：10/10 → 明确了药物名称（硝苯地平缓释片 30 mg，qd；缬沙坦 80 mg，qd）、剂量和用法，符合评分标准，得满分。
  • 非药物治疗：15/15 → 包含了高血压知识指导、生活方式和行为指导、自我保健知识指导、患者家属教育、告知患者紧急情况下的处理措施、具体的体重控制、低盐低脂饮食、戒酒、保持良好心境等行为干预措施，符合评分标准，得满分。
  • 随诊要求：4/4 → 明确了2周后复查血压、血电解质及肾功能，若血压控制不佳需密切随访，符合评分标准，得满分。

处置计划总分 = 6 + 10 + 15 + 4 = 35/35分

【总分计算】

总分 = 25 + 13 + 21 + 35 = 94/100分
